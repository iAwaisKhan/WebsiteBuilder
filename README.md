# Studio — Modern Website Builder

A minimalistic, manual website builder for rapid web design and prototyping.

## Getting Started

### 1. Prerequisites
- **Node.js 18+**

### 2. Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### 3. Running for Development
Launch the builder locally:
```bash
npm run dev
```

## 🛠️ Tech Stack
- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Zustand** (State Management)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Deployment
Build for production:
```bash
npm run build
```
Deploy the `dist/` folder to any static hosting service.
